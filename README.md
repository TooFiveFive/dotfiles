# My Dotfiles

**OS**: Arch Linux

## Installation
*NOTE: Only do this on a new install*
```
cd ~
git clone https://github.com/ericm/dotfiles
mv dotfiles/__move_contents_to_home__/* .
mv dotfiles/.* .config
mv dotfiles/* .config
rmdir dotfiles
```
